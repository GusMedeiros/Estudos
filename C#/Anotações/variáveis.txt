INTEIROS:
short  -  (16 bits)
ushort -  (16 bits, sem sinal)
int    -  (32 bits)
uint   -  (32 bits, sem sinal)
long   -  (64 bits)
ulong  -  (64 bits, sem sinal)

==================================
bool -> true/false (lowercase, diferente de python)
char -> caractere único, assim como em c, usa aspas simples.
byte -> inteiro sem sinal de 1 byte/8bits, ou seja, de 0 a 255.
int -> inteiro usual
long -> inteiro com o dobro do número de bits
float -> float usual
double -> float com o dobro de bits, tendo maior precisão
decimal -> é um número em base 10, o que permite representar acuradamente números sem valores exatos em binário.
string -> string usual, conjunto de caracteres.
